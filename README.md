## ğŸš€ Quick Start Guide

Follow these 3 steps to get running in under 2 minutes.

### 1. Clone the Repository
Open your terminal (Command Prompt or Git Bash) and run:
```bash
git clone [https://github.com/najmularifeen786/claude-artifact-runner.git](https://github.com/najmularifeen786/claude-artifact-runner.git)
cd claude-artifact-runner
```
### 2. Install Dependencies
**Critical Step** This downloads the "engine" that powers the components.
```bash
npm install
```
### 3. Start the Server
Turn on the local development server
```bash
npm run dev
```
You will see a link (usually http://localhost:5173). Ctrl + Click it to open your browser.

## â“ Troubleshooting

**"I see a blank page or white buttons."**
* Did you run `npm install`? If not, stop the server and run it.
* Did you stop the server? Press `Ctrl + C` in the terminal and run `npm run dev` again.

**"Port 5173 is already in use."**
* You might have another project running. Vite will automatically switch to port 5174. Just check the terminal link.

**"The import paths are wrong."**
* This repo uses standard imports. If Claude gives you code with `@/components/ui/...`, it will work automatically because we configured the path aliases for you.
